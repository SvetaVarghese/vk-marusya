# VK Marusya (React)

Проект на React с использованием API VK Marusya.  
Реализован в рамках обучения, с упором на работу с компонентами и логикой приложения.

## Стек
- React
- JavaScript / TypeScript
- HTML, CSS
- REST API

## Функционал
- Работа с API
- Отображение данных в интерфейсе
- Компонентный подход
- Обработка пользовательских действий
